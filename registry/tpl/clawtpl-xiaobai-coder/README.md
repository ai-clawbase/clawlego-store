# 小白程序员 · ClawTpl

面向非技术创业者的全栈编码出厂模板。

ClawTpl 是 **智能体基因 + 出厂模板**：它不是单个资产，而是一整套「人格 + 资产 + 引擎偏好」的声明。在 ClawLego 商店点击安装后，会在你的实例下落地一个完整的编码型 Agent。

## 包含什么

| 类型 | 资产 | 说明 |
|---|---|---|
| Persona | `persona.md` | 系统人格：怎样理解需求、怎样动手 |
| Skill | `skills/code-review` | 代码评审清单 |
| Skill | `skills/scaffold` | 新项目脚手架流程 |
| Subagent | `subagents/debugger.md` | 专职定位 bug 的子智能体 |
| Knowledge | `knowledge/stack-notes.md` | 默认技术栈选型笔记 |

## 安装去向

安装时整包解压到实例文件树的 `business/agents/clawtpl-xiaobai-coder/`。
`agent.json` 是入口清单，ClawLego 读取它来注册这个智能体。
