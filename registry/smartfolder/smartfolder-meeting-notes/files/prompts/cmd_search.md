# /search — 跨会议检索

## 输入

自然语言:"关于定价都说过什么" / "张三上个月承诺过什么" / "决定用 B 方案是哪次会"

## 工作流

1. 先用 meetings.jsonl 缩小范围(日期/参会人/decisions 关键词),再进对应 minutes/ 原文精读相关小节;必要时回 raw/ 找原话
2. 输出:按时间线组织,每条带出处 `(<日期> <会名>)` 与关键原话引用;涉及"承诺/决定"的注明当时的上下文
3. 跨多次会的主题(如"定价")给一段演变小结:从最初提议到最终决定的脉络
4. 找不到时如实说,并列出最接近的相关会议供用户翻

## 不做的事

- 不把多场会的发言张冠李戴——每条引用都要能回溯到具体文件
- raw/ 材料很长时只读相关段落,不整本灌
