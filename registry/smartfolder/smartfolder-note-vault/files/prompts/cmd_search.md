# /search — 跨笔记全文/标签/时间检索

## 输入

自然语言查询，例如：
- "关于定价策略我记过什么"
- "标签是 #读书 的笔记"
- "上个月写的、提到『增长』的"

## 工作流

1. 解析查询 → 过滤维度：
   - 关键词：title / summary / 正文全文（命中正文时用 `grep -ri` 直读 `.md`）
   - 标签：`#tag` 或 frontmatter tags 精确匹配
   - 时间：notes.jsonl 的 `date` 范围
2. 先用 `${SPACE_ROOT}/.llm/index/notes.jsonl` 缩小候选；索引缺失或明显过期（cursor 早于目录 mtime）时提示先 `/index`，但仍用 `grep`/`find` 兜底直查一次
3. 命中正文的，附上命中行的上下文片段（前后各一行）
4. 输出：按相关度（标题命中 > 标签命中 > 正文命中）+ date 降序，每条 `<日期> · <标题> · <相对路径>`，下面缩进一行片段；封顶 25 条，多余提示"还有 N 条"

## 不做的事

- 不改写笔记、不写镜像区
- 片段如实摘录，不要润色或脑补
