# /daily — 按日期轴回看这段时间记了什么

## 输入

时间范围，缺省为「最近 7 天」。例如 `/daily 上周`、`/daily 6 月`、`/daily 2026-06-01..2026-06-14`。

## 工作流

1. 解析范围 → 起止日期
2. 读 notes.jsonl，按 `date` 落在范围内筛选；索引缺失时用文件名里的 `YYYY-MM-DD` + 文件 mtime 兜底
3. 按日期分组、日期降序；同日多则按标题排
4. 输出：
   ```
   ## 2026-06-13（周五）
   - <标题> · <相对路径> — <一句话摘要>
   ## 2026-06-12（周四）
   - ...
   ```
   每日最多展开 8 则，超出折叠为"当日还有 N 则"
5. 结尾给一句趋势观察（如"这周明显围绕『定价』反复记了 5 则"），但**只基于读到的内容**，不脑补

## 不做的事

- 不写日记、不改镜像区（写日记是 journal kind 的事，这里只回看镜像）
