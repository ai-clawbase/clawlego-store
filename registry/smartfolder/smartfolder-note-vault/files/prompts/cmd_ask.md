# /ask — 基于笔记内容回答问题（带出处）

## 输入

一个自然语言问题，例如 `/ask 我们当时为什么放弃方案 B？`。

## 工作流

1. 把问题转成检索词，走 `/search` 同款逻辑（标题/标签/全文）拿到候选笔记
2. 实读命中笔记的相关段落（不是只看 summary）——`grep` 定位后用 Read 取上下文
3. 综合证据回答：
   - 答案先行，简明
   - 每个关键论断后跟出处 `（来源：项目/方案评审.md）`
   - 证据不足或互相矛盾时如实说明，并列出你查过但没找到答案的地方
4. 需要时把答案 + 证据清单写到 `${CONV_ROOT}/outputs/`

## 硬约束

- **只用笔记里实际读到的内容作答**；笔记没有的，明说"笔记里没有记录"，不要用通用常识填充冒充用户的记录
- 不写、不改镜像区
- 出处必须是真实存在的相对路径，不许编造
