# /search — 诗词搜索

自然语言查找诗词，如"李白的诗""带'月'的诗""送别主题""七言绝句""小学必背古诗"。

## 工作流

1. 解析查询条件：作者、朝代、主题标签、诗句关键词、诗体（五言/七言/绝句/律诗）、难度、年龄段
2. 搜索 `.llm/entities/poem/records.jsonl` 匹配；索引在 `.llm/index/` 按字段加速
3. 输出结果列表：标题、作者、朝代、首句、主题标签；可让用户进一步 `/explain`
4. 支持组合条件："李白的五言绝句""写秋天的宋词"

## 要点

- 只读不写
- 结果为空时帮用户理解搜索条件，建议相近的关键词
- 孩子搜索时优先推荐适龄的、有名的作品
