# /quiz — 知识闯关

出题考考孩子的诗词知识。支持"随机出题""关于李白的题""送别诗专题""出 5 道连线题"。

## 出题规则

- 按 `fields.age_range` 调整难度
- 以鼓励和趣味为主，不是考试

## 题型

### 1. 选择题（最常用）
示例："床前明月光"的下一句是？
A. 低头思故乡  B. 疑是地上霜  C. 举头望明月  D. 处处闻啼鸟
→ 孩子选完后解释为什么对/错

### 2. 填空题
示例：举头望____，低头思____。
孩子答完再整句朗读一遍，加深记忆

### 3. 连线题（适合 UI，文字版也可）
将诗人与作品连线："李白—静夜思""杜甫—春夜喜雨"
→ 你是哪个朝代的诗人？→ 唐

### 4. 看图猜诗（需 LLM 画图能力）
用文字描述一幅画面，让孩子猜对应的诗
如"一个老爷爷在河边送朋友坐船走"→ 黄鹤楼送孟浩然之广陵

### 5. 理解题（6 岁以上）
"这首诗里诗人为什么……""如果你是诗人你会……"

## 出题工作流

1. 确定范围：随机 / 某诗人 / 某主题 / 某首诗
2. 生成题目（每轮 3-5 道）
3. 逐题输出，等孩子回答后再出下一题
4. 答完显示成绩和星星评价 ⭐⭐⭐
5. 对错题进行简单讲解，说"这道题有意思，我们来看……"
6. 记录闯关成绩到 `.llm/data/favorites.json` 的 `quiz_history`

## 要点

- 错题不批评，而是："噢，这个有点难，我跟你讲……"
- 连续答对 3 题给彩蛋（一首趣味小诗/诗人的冷知识）
- 可以把错题记录下来，下次优先出
