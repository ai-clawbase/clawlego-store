# /parent-tips — 育儿锦囊

从一首诗出发，给家长提供育儿启发。输入"/parent-tips 静夜思"或"这首诗能教孩子什么"。

## 工作流

1. 查找指定的诗（或从最近讲过的诗提取）
2. 从以下几个角度输出育儿建议：

### 输出结构

**《<诗名>》的育儿锦囊**

🧒 **适龄建议**
- 3-6 岁可以跟孩子聊什么
- 6-12 岁可以深入讨论什么

💬 **亲子对话示例**
- "如果你是诗人，晚上一个人在外面会想什么？"
- "诗里的月亮和我们今天看到的月亮一样吗？"

🌱 **品格教育切入点**
- 思乡→亲情的重要性、感恩家人
- 惜时→"少壮不努力，老大徒伤悲"
- 友情→送别诗的深厚友谊
- 自然→"两个黄鹂鸣翠柳"的观察力
- 坚韧→"墙角数枝梅，凌寒独自开"

🎮 **延伸游戏**
- 角色扮演：演一演诗里的场景
- 找一找：在周围找诗里提到的东西
- 比一比：两首同主题诗有什么不同

📚 **拓展阅读**
- 同作者的其他适合孩子的诗
- 相关成语故事或历史故事

## 要点

- 以建议的口吻，不说教、不制造焦虑
- 每个建议都具体可操作，不是空话
- 强调"和孩子一起"，不是"让孩子做"
