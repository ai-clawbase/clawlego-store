# /daily — 每日一诗

每天推荐一首适合今天读的诗，附带亲子互动小任务。

## 推荐逻辑

- **节气/季节优先**：今天是什么节气/月份 → 推荐相关的诗（清明→清明诗、春天→春晓等）
- **学习进度**：参考最近背过的诗和掌握情况
- **节日相关**：春节/中秋/端午等推荐应节诗
- **新鲜度**：不重复推荐最近 7 天推过的

## 每日输出结构

```
📅 <日期>  |  <节气/季节>

今日推荐：<诗名> — <作者>

📜 原诗
（带拼音）

🎯 今日小任务
1. 读一读：和爸爸妈妈一起读 3 遍
2. 抄一抄：抄写最喜欢的一句
3. 画一画：把诗里的画面画下来
4. 想一想：诗人写这首诗时是什么心情？

💬 亲子对话
- "你觉得这首诗里哪个词最美？"
- "如果你是诗人，你会怎么写？"

⭐ 本周诗单
（列出本周推荐的诗，方便复习）
```

## 记录

- 每日推荐存到 `.llm/data/daily.json`
- 孩子的完成任务情况由家长/孩子口述，你帮忙记录
- 连续完成 7 天给一个小成就称号

## 主动推送

当用户打开对话且当天还没有推荐时，主动问一句："今天还没有读诗呢，要不要看看今天推荐哪首诗？"
