这是一间**智能古诗词书房**——从唐诗三百首到全宋词、诗经、花间集，LLM 帮你给孩子讲解诗意、陪练背诵、出题闯关，甚至引导 TA 创作人生第一首小诗。

所有诗词数据从开源库（chinese-poetry/chinese-poetry）拉取，存在你的文件夹里。文件夹永远属于你。

# 你能用它做什么

- "讲一下静夜思"（/explain 用孩子能懂的语言讲解）
- "帮我出 5 道关于李白的题"（/quiz 闯关模式）
- "陪孩子背春晓"（/recite 逐句跟读 + 填空）
- "以月亮为主题，教孩子写一首小诗"（/compose 小小诗人）
- "今天适合读哪首诗？"（/daily 每日一诗 + 亲子任务）
- "这首诗可以教孩子什么品格？"（/parent-tips 育儿锦囊）
- "搜索所有带'月'的诗"（/search 按主题/作者/朝代查找）
- "从 GitHub 导入全唐诗数据"（/import 拉取开源数据集）

# 文件夹结构

```
<古诗词根>/
├── llm_output/                ← 可见产物(导出、背诵报告、孩子作品)
└── .llm/
    ├── template/              ← 模板:数据契约 schemas/ + seed/ 种子数据
    ├── data/                  ← 你的诗词库:poems/ + authors + categories + favorites + daily
    ├── entities/poem/         ← scoped 实体库(诗词记录)
    ├── index/ · cache/        ← 派生层:索引与解析缓存
    └── ui/                    ← 工作台视图(诗词库/讲解/闯关/创作)
```

# 设计原则

- **诗词数据可追溯**：每首诗标注来源数据集和版本，方便更新
- **年龄自适应**：讲解深度、题目难度、创作引导都按 child_age 调整
- **亲子共读**：每个功能都包含亲子互动建议，不只是给孩子用
- **派生可重建**：索引、排行榜都从实体库实时算，坏了就重建
