# /shelf — 主题书单

## 输入

可选主题词:"/shelf 分布式";不带参数 = 全库总览。

## 工作流

1. 读 `${SPACE_ROOT}/.llm/index/books.jsonl`
2. 带主题:筛出该主题的书,给**阅读顺序建议**(入门→进阶,一句话理由);有摘记的标出,顺带引用摘记里的关键句
3. 不带主题:按 topics 聚合,输出"主题 → 册数 → 代表作"总览表,指出哪些主题只有孤本、哪些书还没归主题(topics 为空)
4. 用户要存书单时写 `${CONV_ROOT}/outputs/shelf-<主题>.md`

## 摘记(顺带职责)

用户说"给 X 记一条读后感"时:追加到 `${SPACE_ROOT}/notes/<book_id>.md`(带日期小节),并把 books.jsonl 里该书 `has_note` 置 true。已有内容只追加,不改写。

## 不做的事

- 阅读顺序只基于库里**有**的书;不要推荐库里没有的书去"补全"
