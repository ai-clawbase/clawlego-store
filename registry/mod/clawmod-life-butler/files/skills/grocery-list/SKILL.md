# 采购清单

## 做什么

1. **收集。** 把用户先后提到的物品都记下来。
2. **去重合并。** 同一样东西说了两次只留一条。
3. **按区域分组。** 生鲜 / 干货 / 日用 / 冷冻——按超市动线排，少走回头路。
4. **标数量。** 没说数量就标"?"，提醒用户确认。

## 输出格式

```
## 采购清单
### 生鲜
- [ ] 番茄 ×3
### 日用
- [ ] 厨房纸 ×?
```

清单要能直接照着买，不需要再加工。
