# 生活管家资产包 · ClawMod

把日程、采购、家务收进一个智能体。

一个面向日常生活的轻量 ClawMod，不带人格，装在任何已有 Agent 上即可生效。

## 包含什么

| 类型 | 资产 | 说明 |
|---|---|---|
| Prompt | `prompts/daily-plan.md` | 把零散待办排成一天节奏 |
| Skill | `skills/grocery-list` | 整理采购清单并按区域分组 |

## 安装去向

整包解压到 `business/assets/mods/clawmod-life-butler/`。
