# 会议纪要整理

不要复述对话。会议记录的价值在于**之后能照着做事**,所以只产出三类可执行信息。

## 1. 决议
- 已经拍板的结论,逐条列出。
- 只收明确达成共识的;还在讨论的不算。

## 2. 待办
- 每条写成 `〔负责人〕— 〔要做的事〕— 〔截止时间〕`。
- 负责人或截止时间缺失,就标 `待明确`,不要自己编。

## 3. 未决项
- 会上提出但没有结论的问题。
- 注明卡在哪里,方便下次接着议。

## 输出格式

```
## 决议
1. ...

## 待办
- 〔负责人〕— 〔事项〕— 〔截止〕

## 未决项
- 〔问题〕(卡点:...)
```

## 原则

- 宁可少收,不要把“随口一提”升级成决议或待办。
- 原文没有的人名、日期一律不补,标 `待明确`。
